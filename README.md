# subnet-ui-katya

Это моя библиотека компонентов для Vue 3.  
Я сделала её, чтобы удобно собирать красивые и аккуратные формы — например, для калькулятора подсетей.

Все элементы выдержаны в едином стиле:  
- Светлый фон  
- Мягкие закругления  
- Голубые и фиолетовые акценты (как неоновое свечение)


Этот компонент был создан в образовательных целях — для изучения работы с Vue 3, TypeScript и scoped-стилями.

## Что внутри?

В библиотеке 4 готовых компонента:

- **`UiButton`** — кнопка  
  Есть основная - с градиентом, и второстепенная - с рамкой. Кнопки можно отключить.

- **`UiInput`** — поле для ввода текста  
  Подсвечивается при клике, можно сделать недоступным.

- **`UiSelect`** — выпадающий список  
  Выглядит так же, как поле ввода, и тоже можно заблокировать.

- **`UiField`** — рамка с надписью сверху  
  Внутрь можно вставить любое поле: инпут, селект или даже что-то своё.

Все компоненты работают с `v-model`, как обычные Vue-элементы.
